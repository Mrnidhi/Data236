# DATA-236 Homework 2

## Part 1 - Artist Liberty (HTML & CSS)
A responsive artist portfolio website built with pure HTML and CSS, featuring a hero section, about section, artwork gallery with hover effects, contact form, and mobile-responsive layout.

**Files:** `artist-liberty/index.html`, `artist-liberty/styles.css`

## Part 2 - Book Management App (FastAPI)
A FastAPI web application with Jinja2 templates to manage an in-memory list of books. Supports adding, updating (Book ID 1), deleting (highest ID), and searching by title.

**Files:** `fastapi-books/main.py`, `fastapi-books/templates/`

## Part 3 - Stateful Agent Graph (LangGraph)
A stateful agent graph using LangGraph with Planner, Reviewer, and Supervisor nodes following the Hub-Spoke routing pattern. Includes a correction loop and turn-count termination.

**File:** `agent-graph/agent_graph.py`
